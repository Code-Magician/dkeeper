# DKeeper
#### A Note Keeping app same as google Keep Made with Dfinity Blockchain as a backend and React.js as a frontend.

## Instructions to Run the Project

#### To start the project run below command in one Terminal. ```dfx start```
![image](https://github.com/Code-Magician/dkeeper/assets/74300477/c34eed40-5dc7-4d2f-a014-42b15bb64d1c)

#### Then deploy the Dfinity Blockchain Backend using below command in another terminal. ```dfx deploy```
![image](https://github.com/Code-Magician/dkeeper/assets/74300477/48de7de7-a582-4fee-81a3-7f3179581d9c)

#### Them start the React frontend using the below command. ```npm start```
![image](https://github.com/Code-Magician/dkeeper/assets/74300477/f8dc4d26-54c7-4a8d-9ba0-8edb809dd4af)
